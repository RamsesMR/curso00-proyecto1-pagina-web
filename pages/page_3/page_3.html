INSTRUCCIONES QUE TIENEN QUE VER CON LAS FUENTES

FUENTES DE ASPECTO

La etiqueta "font" es usada para modificar el tamaño, el color y la fuente del texto

FONT FACE

Se utiliza para cambiar el tipo de fuente
busca entre tres tipos de fuente diferente si no las reconoce
Se utiliza en la parte de la estructura de BODY

FONT COLOR

Se utiliza para poner sólo el texto que queremos poner en color.
El color se especificaba con su nombre o con su código hexadecimal.

FONT SIZE

Especifica la dimensión de la letra

MÁS ETIQUETAS DE ASPECTO

BODY TEXT

Se usa para fijar las características del texto

BODY:LINK,VLINK Y ALINK

link: es el color de los enlaces que no se han visitado
vlink: es atributo de color de los enlaces que ya hemos visitados
alink: es el color de los enlaces cuando pulsamos sobre ellos.

BODY background-color

Para especificar el color de fondo de la página.
El color se define en base hexadecimal de la siguiente forma:
(#rrggbb) en el orden rojo, verde, azul (Red, Green, Blue).
Es decir, cada color se define con el carácter # seguido de 6 letras ó números.

HTML utiliza códigos especiales para admitir contenido de símbolos comunes que no se encuentran en los teclados, así como en situaciones en las que el símbolo en sí tiene un doble propósito en HTML como carácter de control y carácter de visualización.Aquí tienes el link pulsar aquí
