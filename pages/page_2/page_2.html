"LISTAS EN HTML"

CLASES DESCRIPCION TIPOS
Listas Ordenadas
OL
Son aquellas que se ordenan mediante numeración 1,a,A,i,I
Listas Desordenadas
UL
Son aquellas que se ordenan mediante viñetas disc,square,circle,none
En HTML, podemos crear una
  1. LISTA ORDENADA usando la etiqueta OL.
    • La etiqueta ol representa una lista ordenada.
    • Dentro de cada uno de los elementos de la lista ordenada inicio ol y final ol,tenemos que definir los elementos de la lista.
    • Podemos definir los elementos de la lista usando la etiqueta li.
    • Dentro de una lista ordenada podemos crear más subniveles.
  2. LISTA DESORDENADA usando la etiqueta UL.
    • La etiqueta ul representa una lista desordenada.
    • Para representar los elementos de la lista desordenada utilizamos el mismo elemento que con las listas ordenadas, es decir, el elemento li.
    • Dentro de una lista desordenada podemos crear más subniveles.
